Pokédex Flutter Challenge

This is a Flutter challenge to create a Pokédex app using the [PokéAPI](https://pokeapi.co/).

## Getting Started

1. Clone this repository
2. Run `flutter pub get`
3. Run `flutter run`
4. Enjoy!
